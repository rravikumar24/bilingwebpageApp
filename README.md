# FieldDesk - Sales & AMC Entry

A web-based billing and entry management page for tracking Sales and AMC (Annual Maintenance Contract) records.

## Features

- Sales and AMC data entry forms
- Sidebar navigation
- Responsive layout

## Tech Stack

- HTML, CSS, JavaScript
- Select2 for dropdowns
- Font Awesome icons
- Google Fonts (Poppins)

## Usage

Open `index.html` in a browser to get started.
